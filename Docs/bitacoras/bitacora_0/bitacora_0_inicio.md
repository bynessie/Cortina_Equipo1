# Bitácora 0 – Inicio del Proyecto

## Comprensión del problema
Diseño e implementación de un sistema mecatrónico para el control automático del enrollado y desenrollado de una cortina industrial, incorporando la configuración de parámetros como tiempo, altura y velocidad, así como criterios de seguridad operativa y una interfaz de usuario estructurada.

## Retos percibidos
El mayor reto del proyecto será la integración adecuada del sistema con la interfaz de usuario, ya que es fundamental que las alarmas se muestren de manera clara para el operador y que además queden registradas correctamente para su posterior análisis y mantenimiento.

## Expectativas de aprendizaje
Al finalizar el curso, esperamos adquirir una metodología clara de diseño mecatrónico que nos permita analizar requerimientos e integrar correctamente los distintos subsistemas del proyecto. Buscamos fortalecer nuestras habilidades en el cálculo y selección de actuadores, así como en la programación en LOGO, implementando máquinas de estados, temporizadores y rutinas de seguridad.

Asimismo, esperamos desarrollar interfaces HMI claras y funcionales, con navegación jerárquica y visualización de parámetros en tiempo real, y reforzar el trabajo en equipo mediante una adecuada documentación técnica, validando las soluciones a través de simulaciones o pruebas.
